Location : Pennsylvania State University
Dates Attented: January 2016 – March 2016

Using WAMP Server, I created a web user interface for the Global Lion Mentor program at Penn State. Via this interface, students could register either as a mentee or mentor and interact with other students on campus. This program is a mentoring program where existing Penn State students can help incoming international students acclimatize to the new surroundings.

  • Users can send messages, view campus events, and view the list of other users registered in the website.
  • Users also have their profile page where edits can be made.
  • Admin has a special privilege where they can add events, pair-up mentors and mentees.
  • Links to important information (such as health insurance links, etc) are posted by the admin.
  • WAMP: Windows, Apache, MySQL, PHP
  • HTML, CSS, JavaScript

Location : Pennsylvania State University
Dates Attented: January 2015 – March 2015

Requirements for a Web User Interface were determined for 'Kick in for Kids', a charitable organization in Central Pennsylvania who help students & families going through temporary hardship. The project's focus was the effective collection of user analysis and requirements to build a web user interface.

  • Survey forms were coded in HTML5, and were sent to the board of the organization for their valuable input.
  • Upon submission, the form would be sent to the "kick in for kids' gmail account.
  • This was achieved by using the WAMP server and making changes to the "send_mail.ini" file.
  • Once a design was fixed, the web user interface was created using HTML5, CSS and PHP.
  • WAMP: Windows, Apache, MySQL, PHP
  • HTML, CSS, JavaScript

Created a database for a local restaurant, Brownstone Cafe, in Middletown, P.A. Firstly, an ERD was created in Microsoft Visio for the design of the database.
